# Data Analysis Portfolio

This repository contains a collection of data analysis projects demonstrating my approach to solving business problems using data.

## About This Repository

The projects showcase my ability to transform raw data into actionable insights. Each project follows a structured process: data understanding, cleaning, analysis, and result presentation.

I view data analysis as a bridge between raw data and business decisions. My methodology includes four key steps:

1. **Data cleaning and validation**: Ensuring data accuracy and addressing errors or missing values before analysis
2. **Data transformation**: Structuring raw data to answer specific business questions
3. **Analysis**: Identifying trends, patterns, and key performance indicators
4. **Insights and reporting**: Communicating results clearly through visualizations and summaries to support decision-making

## Technical Skills

I work with SQL, BigQuery, Google Sheets, and Python for data analysis. I prioritize writing efficient queries and creating clear reports that are accessible to both technical and non-technical stakeholders.
